# ДЗ - Сравнение методов классификации

Обучил и сравнил 5 моделей из sklearn

## Модели

- KNN
- Decision Tree
- Random Forest
- Logistic Regression  
- SVM

## Что сделано

- генерация синтетических данных
- разделение на train/test
- нормализация
- обучение 5 моделей
- сравнение по accuracy
- матрицы ошибок для всех моделей
- детальный отчет для лучшей модели

## Запуск

```bash
jupyter notebook klassifikatsiya.ipynb
```
